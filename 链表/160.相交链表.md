双指针法

假设链表 A 的不相交部分长度为 $a$，
链表 B 的不相交部分长度为 $b$，
两个链表相交的公共部分长度为 $c$。
链表 A 的总长度是 $a + c$
链表 B 的总长度是 $b + c$
如果我们使用两个指针 pA 和 pB 分别从 headA 和 headB 开始遍历：
    消除长度差：
         当 pA 遍历完链表 A 时，我们让它跳转到链表 B 的头部继续遍历；
         同理，当 pB 遍历完链表 B 时，我们让它跳转到链表 A 的头部。
    相遇点即为交点： 
        这样，pA 走过的总路程将是 $a + c + b$，而 pB 走过的总路程将是 $b + c + a$。
        因为 $a + c + b = b + c + a$，所以无论两个链表原来有多长，两个指针最终一定会同时到达相交节点。
    如果不相交： 
        如果两个链表没有交点（即 $c = 0$），那么 pA 和 pB 都会在走了 $a + b$ 步之后，同时到达各自最后跳转链表的末尾（皆为 None），此时循环结束，返回 None。

```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        if not headA or not headB:
            return None

        pA, pB = headA, headB

        while pA != pB:
            pA = pA.next if pA else headB
            pB = pB.next if pB else headA

        return pA

```
