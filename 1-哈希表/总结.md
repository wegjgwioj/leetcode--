**专题总结**
`1-哈希表` 这部分你其实已经刷到了哈希题最核心的 3 个母题：

- 查找型: [1-两数之和.py](c:/Users/31431/Desktop/项目/1-哈希表/1-两数之和.py)
核心是“边遍历边查表”，把 `O(n)` 查找变成 `O(1)` 查找。
模板是：
```python
seen = {}
for i, x in enumerate(nums):
    need = target - x
    if need in seen:
        return [seen[need], i]
    seen[x] = i
```

- 分组型: [49-字母异位词分组.py](c:/Users/31431/Desktop/项目/1-哈希表/49-字母异位词分组.py)
核心是“构造统一 key，再按 key 分桶”。
模板是：
```python
groups = {}
for word in strs:
    key = ''.join(sorted(word))
    groups.setdefault(key, []).append(word)
return list(groups.values())
```

- 连续性/去重型: [128-最长连续序列.py](c:/Users/31431/Desktop/项目/1-哈希表/128-最长连续序列.py)
核心是“先放进 `set`，再做快速存在性判断”。
标准思路是：
```python
num_set = set(nums)
for x in num_set:
    if x - 1 not in num_set:
        # 从起点开始扩展
```

**本质**
哈希表专题就一句话：`用空间换时间`。

- `dict` 解决“我想快速知道某个值对应什么”
- `set` 解决“我想快速知道某个值在不在”

所以现在可以把哈希题归成这 4 类：

- 是否存在: `x in seen`
- 映射关系: `seen[x] = i`
- 计数统计: `cnt[x] += 1`
- 分组归类: `groups[key].append(x)`

**你现在该记住的 4 个模板**
```python
# 1. 查找补数
seen = {}
for i, x in enumerate(nums):
    if target - x in seen:
        ...
    seen[x] = i
```

```python
# 2. 频次统计
cnt = {}
for x in nums:
    cnt[x] = cnt.get(x, 0) + 1
```

```python
# 3. 分组
groups = {}
for x in data:
    key = ...
    groups.setdefault(key, []).append(x)
```

```python
# 4. 集合判重 / 连续判断
s = set(nums)
if x in s:
    ...
if x - 1 not in s:
    ...
```

**容易错的点**
- 把 `list` 当哈希表用。`list` 适合按下标访问，不适合按值快速查找。
- `dict` 的 key 需要“稳定”。49 题里不能直接用原字符串，要用排序后的字符串。
- 128 题里如果走排序扫描，必须先去重；如果走 `set`，必须只从起点开始扩展，不然会重复算。
- 别把“次数”当“长度”。这是你前面在 128 题里碰到过的点。

**怎么判断一题该不该用哈希**
看到这些信号就优先想 `dict/set`：

- “快速判断某值是否出现过”
- “快速找到另一个配对值”
- “统计出现次数”
- “按某种特征分组”
- “去重”
- “防止重复访问”
- “把 `O(n^2)` 优化成 `O(n)`”

**一句话总纲**
哈希表专题的学习顺序可以记成：

`查存在 -> 做映射 -> 统计频次 -> 按 key 分组 -> 配合 set 判连续 -> 和前缀和/滑动窗口结合`
