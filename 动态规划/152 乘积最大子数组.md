核心思想：动态规划（双状态转移）
状态定义：

max_so_far：以当前元素结尾的连续子数组的最大乘积。

min_so_far：以当前元素结尾的连续子数组的最小乘积。

状态转移：

当遍历到当前元素 curr 时，如果 curr 是负数，那么乘以负数会导致大小反转。因此，我们遇到负数时，可以直接交换 max_so_far 和 min_so_far。

接着，我们要决定是把当前元素加入之前的连续子数组中，还是让当前元素“自立门户”成为一个新的起点。

max_so_far = max(curr, max_so_far * curr)

min_so_far = min(curr, min_so_far * curr)

结果更新：在每一步中，都将全局最大结果 res 与更新后的 max_so_far 进行比较并保留最大值。

"负负得正"：
```python

if __init__=="__main__":
    nums = [2,3,-2,4]
    res=dpmax=dpmin = nums[0]

    for i in range(1,len(nums)):
        curr = nums[i]
        if curr<0:
            dpmax,dpmin =dpmin,dpmax
        dpmax=max(curr,dpmax*curr)
        dpmin=min(curr,dpmin*curr)

        res = max(res,dpmax)
    
    print(res)
```
