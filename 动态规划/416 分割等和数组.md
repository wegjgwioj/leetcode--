```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        # sum/2 == 2*(sum/2)
        if not  nums:
             return False
        if len(nums)==1 :
            return  False

        mysum=sum(nums)
        if mysum%2 !=0:
             return False
        target = mysum // 2
        # 能不能凑出对应数值
        dp=[False]*(target+1)
        dp[0]=True

        for num in nums:
            for j in range(target,num-1,-1):
                dp[j] = dp[j] or dp[j-num]

        return dp[target]
```

状态定义：构建一个布尔型数组 dp，其中 dp[j] 表示能否从数组中挑出几个数字，使得它们的和正好等于 j。

初始状态：dp[0] = True。意思是，如果目标和是 0，我们一个数字都不挑就能满足，所以是可行的。

状态转移：对于数组里的每一个数字 num，我们要更新 dp 数组。
如果当前的容量 j 装得下 num，我们就有两种选择：

不把 num 放进去，那么能不能凑齐容量 j 就看之前的结果 dp[j]。

把 num 放进去，那么能不能凑齐容量 j 就看之前能不能凑齐容量 j - num。
所以公式是：dp[j] = dp[j] or dp[j - num]。
