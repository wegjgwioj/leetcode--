coins[] int  不同面额的硬币
amount int  金额总数

计算：可以凑成 amount 的组合数个数最小，否则返回-1
n=coins.length
$curramount = coins[0]*k_0+ coins[1]*k_1 + ... + coins[n-1]*k_{n-1} $
==> $k_0+k_1+...+k_{n-1} = \text{最小的组合数}$
# DP

```python
import math


if __name__ == "__main__":
    coins =[1,2,5]
    amount = 11
    dp = [math.inf] * (amount + 1)
    dp[0] = 0
    for i in range(1, amount + 1):
        for coin in coins:
            if i - coin >= 0:
                dp[i] = min(dp[i], dp[i - coin] + 1)
    print(dp[amount] if dp[amount] != math.inf else -1)

    # dp[i] 代表什么？ 它代表凑成金额 i 所需要的最少硬币个数



```
