# ​杨辉三角
  左对齐：
[1]
[1,1]
[1,2,1]
[1,3,3,1]
[1,4,6,4,1]
​
返回值二维数组c
c[i][0] = 1
c[i][i] = 1
c[i][j] = c[i-1][j-1] + c[i-1][j]  ===>组合数恒等式

```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        c = [[0] * (i + 1) for i in range(numRows)]
        for i in range(numRows):
            c[i][0] = 1
            c[i][i] = 1
            for j in range(1, i):
                c[i][j] = c[i - 1][j - 1] + c[i - 1][j]
        return c
```
